Описание программы «Training Planner»
Автор: Дмитрий Котов, 2026

Краткое описание программы
Программа «Training Planner» — это простое графическое приложение на Python, предназначенное для планирования и учета тренировок. Пользователи могут добавлять сведения о тренировках (дата, тип, длительность), просматривать их в удобной таблице, фильтровать по типу и дате, а также сохранять и загружать данные в формате JSON. Это помогает легко организовать и отслеживать тренировочный процесс.

Инструкция по запуску
Требования и зависимости:
Python 3.6 или выше (стандартная установка — ничего дополнительно устанавливать не нужно, так как используется стандартная библиотека tkinter)
tkinter (обычно входит в стандартную поставку Python, проверьте установлено ли у вас)
Шаги установки и запуска:
Скачайте или скопируйте файл с исходным кодом программы и сохраните его, например, как training_planner.py.
Откройте командную строку или терминал.
Перейдите в папку с файлом:
bash

cd путь/к/вашему/файлу
Запустите программу командой:
bash

python training_planner.py
Примеры использования и работы программы
Добавление тренировки:
Введите дату в формате YYYY-MM-DD, укажите тип тренировки (например, «Кардио») и длительность в минутах. Нажмите кнопку «Добавить тренировку». Запись появится в таблице.

Просмотр тренировок:
Все добавленные тренировки отображаются в таблице ниже формы.

Фильтрация:
В выпадающем списке выберите тип тренировки (или «Все» для отображения всех), введите дату, если нужно, и нажмите «Применить фильтр». В таблице отобразятся только подходящие записи.

Сброс фильтра:
Нажмите «Сбросить фильтр», чтобы снова увидеть все тренировочные записи.

Сохранение данных:
Нажмите «Сохранить в JSON», выберите файл и сохраните текущие данные.

Загрузка данных:
Нажмите «Загрузить из JSON», выберите файл, и все сохранённые тренировки появятся в таблице.

Это делает программу удобным инструментом для личного учета тренировок, который легко расширять и дополнять!
